# 🎸 Digital Guitar Tuner (Web-Based)

A functional, minimal, and mobile-responsive guitar tuner built with HTML, CSS, and JavaScript.

### 🎶 Project Overview
This tool helps guitarists tune their instruments by playing high-quality reference tones for each string (**E, B, G, D, A, E**). It’s designed with a dark, modern interface optimized for mobile usage.

### ✨ Features
* **Interactive UI:** Large, easy-to-tap buttons for guitar strings.
* **Glow Effects:** Modern aesthetic with neon-style visual feedback.
* **Pure Web App:** No installation needed, runs directly on any smartphone browser.
* **Built on iPhone XR:** Proof of complex functionality developed entirely on mobile.
